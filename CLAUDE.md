# CLAUDE.md

## プロジェクト概要
個人フットサル・個人ソサイチの検索・予約・決済MVP

## 技術スタック
- Next.js (App Router) + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (PostgreSQL + Auth)
- Stripe (決済)
- Vercel (ホスティング)

## デザイン方針

### ブランドカラー
- メインカラー: オレンジ (#F97316 系、F-Channelブランドカラー)
- 背景: 白ベース
- テキスト: ダークグレー

### トーン
- スポーティにしない。イベント検索サイト＋情報発信メディアのイメージ
- 清潔感、モダン、余白たっぷり
- 信頼感と使いやすさを重視

### 参考サイト
- NEWT (newt.net): 白ベース、カード型レイアウト、余白の取り方、CTAボタンのデザイン
- Plab Football (plabfootball.com): 日付カレンダー横スクロール、タグフィルター、時間順リスト、ステータス表示（締切・募集中）

### UIの重要ポイント
- 日程・地域・カテゴリで直感的に検索できること
- 一覧画面で空き状況・特徴・時間がぱっと見でわかること
- 申込から決済までのステップを最短にすること
- スマホファースト（レスポンシブ対応）

### コンポーネント
- shadcn/uiを基本とする
- カスタムCSSはTailwind CSSのユーティリティクラスで対応

## 開発ルール
- TypeScriptの型定義を必ず行う
- コンポーネントは小さく分割する
- 日本語UIで開発する
